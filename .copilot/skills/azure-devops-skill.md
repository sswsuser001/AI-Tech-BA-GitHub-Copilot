# Repository Skill

## Purpose
Help GitHub Copilot Chat understand the repository’s documentation and helper script patterns.

## Key points
- The repository focuses on business requirements, documentation, and lightweight helper scripts.
- Use docs and folder structure as the primary source of truth.
- Keep new automation narrow and documented.

## Typical tasks
- Review documentation structure
- Suggest improvements to requirements artifacts
- Document script usage clearly
