# Copilot Agents

This folder holds agent definitions and documentation for GitHub Copilot Chat workflows.

- `copilot/agents/` — agent metadata and guidance files.
- `copilot/skills/` — skill descriptions or architecture notes for custom tool integration.
- `copilot/tools/` — reusable scripts and tools referenced by Copilot agents.

Use this structure to store Copilot-specific artifacts separately from project source and docs.
